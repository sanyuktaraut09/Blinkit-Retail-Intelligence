# 🛒 Blinkit Retail Intelligence Dashboard

An end-to-end Retail Analytics project built using **Python, MySQL, SQL, Power BI**, with **Kafka Streaming (coming soon)**.

---

## 📌 Project Overview

This project analyzes Blinkit retail sales data to uncover business insights such as:

- Sales Performance
- Product Category Analysis
- Outlet Performance
- Customer Ratings
- Inventory Distribution
- Business KPIs

---

## 🚀 Tech Stack

- Python
- Pandas
- NumPy
- MySQL
- SQL
- Power BI
- Git & GitHub
- Apache Kafka (In Progress)
